Overview:

A simple voice-to-text web app that converts spoken words into text using the Web Speech API. Includes mic glow animation and optional stop command.

Features

One-click voice input
Live transcript display
Auto-stop with custom trigger phrase (optional)
Mic pulse animation during listening
Error handling for unsupported device

Usage

Upload or add song files to the playlist.
Click play OR use voice command.
Enjoy the animated audio visualizer
